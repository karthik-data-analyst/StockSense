# StockSense 🛒
### AI-Powered Restock Assistant for Small Shops

## What is StockSense?
StockSense is an AI tool that analyzes daily 
sales data and tells shop owners exactly what 
to restock and how many units to order.

## The Problem it Solves
Small shop owners in Tamil Nadu spend 30+ minutes 
every morning manually checking stock and deciding 
what to order. They often overstock slow items or 
run out of fast moving ones — costing them money 
every day.

## How it Works
1. Shop staff updates one Excel file daily
2. StockSense reads the sales and stock data
3. AI analyzes and gives clear restock recommendations
4. Owner knows exactly what to order in seconds

## Sample Output

StockSense Daily Report — 21 June 2026

Laddu: order 50 units (only 6 left, selling 31/day)

Murukku: order 40 units (only 8 left, selling 38/day)

Chips: no action needed — stock healthy

Priority: Order Laddu first — runs out before evening
